活动记录 简介
该系统全称：

工作室活动记录管理系统

Studio activity record management system

此系统的目的：

此系统主要用于工作室活动的管理，总体任务是实现工作室活动的记录，规范化和自动化。用计算机对工作室活动日常管理。

该系统的功能能够实现对于活动管理的相关要求，同时各项功能的操作要求简单，不需过多的对人员的培训。

此项目主要面向功能：

1、 输入记录。

对每个组织活动人员进行输入并存储。

2、 维护记录。

能够实现添加、查看活动记录，但是一定要将最终数据存储到文件中。

3、 查询记录。

根据时间、活动内容、姓名等查询活动的详情。

4、 统计记录。

完成对活动的统计。

项目背景

  工作室活动记录管理系统主要是对于工作室的活动做出相对应的整理，后面不管是拿出来回忆大学的生活还是面向下一届的招生都是美好的记忆。

  通过工作室活动管理系统给工作室做活动记录的管理者减轻工作的负担，同时这个系统还可以减少人力与管理时间，提供信息准确度更高更简便的工作室活动管理系统。
